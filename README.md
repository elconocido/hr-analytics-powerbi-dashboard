# HR Analytics Dashboard | Power BI

An interactive HR Analytics dashboard built in **Power BI** to help HR managers monitor workforce performance, identify attrition trends, and make data-driven decisions.

---

## Project Overview

This project transforms raw HR data into meaningful business insights through data cleaning, data modeling, DAX calculations, and interactive visualizations.

The dashboard helps answer key HR questions such as:

- Which departments experience the highest employee attrition?
- Does overtime contribute to employee turnover?
- Which employee groups are most likely to leave?
- How satisfied are employees with their jobs?
- How long do employees wait before promotion?

---

## Business Objective

The objective of this project was to build an executive HR dashboard that enables decision-makers to monitor employee performance, understand attrition patterns, and identify opportunities to improve employee retention.

---

# Dataset

**Dataset:** IBM HR Analytics Employee Attrition & Performance

The dataset contains employee demographic information, compensation, performance, education, promotion history, and attrition records.

---

# Tools Used

- Microsoft Excel
- Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)

---

#  Project Workflow

## 1. Data Preparation

Performed data cleaning using Power Query by:

- Removing duplicate records
- Checking missing values
- Correcting data types
- Standardizing categorical values
- Renaming columns
- Creating Age Groups
- Creating Salary Bands
- Creating Years at Company Groups

---

## 2. Data Modeling

Designed a **flat-schema-inspired** data model to improve reporting performance and simplify analysis.

---

## 3. DAX Measures

Created measures including:

- Total Employees
- Total Attrition
- Attrition Rate
- Average Age
- Average Monthly Income
- Average Job Satisfaction
- Average Work-Life Balance
- Average Years Since Last Promotion
- Average Training Time Last Year

---

## 4. Dashboard Development

Built three interactive dashboard pages:

### Workforce Overview
Provides an executive summary of workforce metrics.

### Attrition Analysis
Analyzes employee turnover and identifies retention risks.

###  Employee Performance
Evaluates employee satisfaction, training, promotion, and work-life balance.

Interactive features include:

- KPI Cards
- Slicers
- Cross-filtering
- Drill-down analysis
- Executive narrative

# Key Insights

- The organization employs approximately **1,470 employees**.
- Overall attrition rate is **16.12%**.
- Research & Development recorded the highest employee turnover.
- Employees working overtime experienced higher attrition.
- Employees aged **25–34** showed the highest turnover.
- Lower salary bands experienced higher attrition.
- Average promotion waiting time is **2.19 years**.
- Most employees received a performance rating of **3**.

---

# Recommendations

- Review overtime policies to reduce burnout.
- Strengthen retention strategies in high-turnover departments.
- Reassess compensation for lower salary bands.
- Expand employee development and engagement programs.
- Improve promotion planning and career progression.
